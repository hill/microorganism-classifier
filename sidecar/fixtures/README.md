# Dev fixtures

Test footage used by the fake-camera dev mode, so you can develop the capture
pipeline without the EP2M plugged in. See `MICRO_DEV_VIDEO` in `frame_source.py`.

## dev_microscopy.mp4

A 26 second clip of swimming paramecia under a brightfield microscope, trimmed
and re-encoded to h264 from the source below. Chosen because it has the circular
eyepiece vignette and the multiple drifting organisms the tracker needs.
